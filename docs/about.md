# About

omnipath-utils is part of the [OmniPath](https://omnipathdb.org) ecosystem,
developed at [Saez Lab](https://saezlab.org) (Heidelberg University Hospital).

It provides the core identifier translation, taxonomy resolution, and
reference list functionality used by OmniPath and related tools.
